# 支持以下基本特性
1. 支持scss，compass
2. 支持cmd，amd 模块化开发
3. 支持coffee-script 
4. 支持前端模板 handbars
5. 上线打包压缩md5

# 基本命令
+ `fis3 server start` 开启静态服务器
+ `fis3 server open` 打开fis 服务目录
+ `fis3 release -wL` (fis3 release -h 查看帮助) 
+ *注意：在本地任意文件夹下 运行 fis3 relase 自动把文件发布上本地静态服务器WWW目录下*
